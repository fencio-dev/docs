## Architecture

An AARM system consists of six components working together:
```
┌─────────────────────────────────────────────────────────────┐
│                       AARM SYSTEM                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌──────────────────┐      ┌──────────────────┐            │
│   │ Action Mediation │ ───► │ Policy Decision  │            │
│   │ Layer (AML)      │      │ Point (PDP)      │            │
│   └──────────────────┘      └────────┬─────────┘            │
│                                      │                      │
│                                      ▼                      │
│                             ┌──────────────────┐            │
│                             │ Policy Enforce-  │            │
```

## Data Flow

<Steps>
  <Step title="Intercept">
    Action Mediation Layer captures tool invocation from agent
  </Step>
  <Step title="Normalize">
    AML converts protocol-specific request to AARM action schema
  </Step>
  <Step title="Evaluate">
    Policy Decision Point matches action against rules
  </Step>
  <Step title="Enforce">
    Policy Enforcement Point implements decision (allow/deny/modify/step-up)
  </Step>
  <Step title="Approve">
    If STEP_UP, Approval Service routes to human approvers
  </Step>
  <Step title="Execute">
    If allowed, action forwards to tool
  </Step>
  <Step title="Record">
    Receipt Generator creates signed audit record
  </Step>
  <Step title="Export">
    Telemetry Exporter sends events to SIEM/SOAR
  </Step>
</Steps>

## Component Responsibilities

| Component | Input | Output | Required |
|-----------|-------|--------|----------|
| Action Mediation | Protocol request | AARM Action | Yes |
| Policy Decision Point | Action | Decision | Yes |
| Policy Enforcement Point | Action + Decision | Enforced result | Yes |
| Approval Service | Action | Approval result | Yes |
| Receipt Generator | Action + Decision + Result | Signed receipt | Yes |
| Telemetry Exporter | Receipt | SIEM events | Recommended |
